# CAP Skill for OpenClaw

### What is OpenClaw

OpenClaw is an agent execution client that allows users to run on-chain agents and connect them to external infrastructure for automated actions such as wallet operations and trading.

### What is an OpenClaw Skill

An OpenClaw Skill is a modular capability that extends what an agent can do. Skills integrate external services into the agent through API connections.

### CAP Skill for OpenClaw

The CAP Skill allows OpenClaw agents to use CAP services via a [CAP API Key](product-features/api-key.md).

By adding a CAP API Key, your agent can access wallets, execute trades, and deploy contracts directly through CAP.

```
Install skill https://raw.githubusercontent.com/Capminal/openclaw-skills/main/capminal/SKILL.md
```
